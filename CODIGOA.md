# Importancia de Git

Git es una herramienta fundamental en el desarrollo moderno porque permite llevar un **control de versiones distribuido**. Esto significa que cada desarrollador tiene una copia local del historial completo del proyecto.

Su uso es vital porque:

- Permite recuperar versiones anteriores del código si algo sale mal.
- Facilita el trabajo fuera de línea.
- Ofrece un registro histórico de quién hizo qué cambio y por qué.
